# side-stacker-game

## stack

- front-end : React, Webpack
- back-end : Flask, Sqlite3

## Installation

- front-end
  ```
  # move to client directory
  ==========================
  yarn install
  ```

- back-end
  ```
  # move to server directory
  ==========================
  pip install -r requirements.txt
  ```

## Run

- front-end
  ```
  # for dev
  yarn start

  # for build
  yarn build

  # for lint
  yarn lint

  # for check prettier
  yarn format
  ```

- back-end
  ```
  flask run
  ```
